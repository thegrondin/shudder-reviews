## Prerequisites

You need to install [**pnpm**](https://pnpm.io/installation)

## Development

```bash
npm install -g pnpm
```

```bash
pnpm i
```

### Running this project

**Development**

*Firefox*

```bash
  pnpm app:dev:firefox
```

*chrome*

```bash
  pnpm app:dev:chrome
```

*edge*

```bash
  pnpm app:dev:edge
```
